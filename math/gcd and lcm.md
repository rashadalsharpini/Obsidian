## gcd {Greatest Common Divisor}
24, 36

| 2   | 24  |     | 2   | 36  |
| --- | --- | --- | --- | --- |
| 2   | 12  |     | 2   | 18  |
| 2   | 6   |     | 3   | 9   |
| 3   | 3   |     | 3   | 3   |
|     | 1   |     |     | 1   |
$$24 = 2^3 * 3$$
$$36 = 2^2 * 3^2$$
$$gcd(24,36) = 2^2 * 3 = 12$$
---
## lcm {Least Common Multiplier}
24, 180

| 2   | 24  |     | 2   | 180 |
| --- | --- | --- | --- | --- |
| 2   | 12  |     | 2   | 90  |
| 2   | 6   |     | 3   | 45  |
| 3   | 3   |     | 3   | 15  |
|     | 1   |     | 5   | 5   |
|     |     |     |     | 1   |
$$24 = 2^3 * 3$$
$$180 = 2^2 * 3^2 * 5$$
$$2^3 * 3^2 * 5 = 360$$
---
## Euclidean Algorithm
$$m*n = gcd(m, n) * lcm(m, n)$$
$$28*52 = lcm(28, 52)* gcd(28, 52)$$
