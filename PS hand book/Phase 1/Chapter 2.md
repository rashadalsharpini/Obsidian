| ---------- | ------------------------ |
| n <= 10    | O($n!$)                  |
| n <= 20    | O($2^n$)                 |

| n is large | O(1) or O($logn$)        |
### Maximum sub-array problem
#### algorithm 1 $n^3$
```cpp
int best = 0;
for(int i = 0;i<n;++i){
	for(int j = i;j<n;++j){
		int sum = 0;
		for(int k = a; k < b; ++k){
			sum += array[k];
		}
		best = max(best, sum);
	}
}
cout<<best<<endl;
```
#### algorithm 2 $n^2$
```cpp
int best = 0;
for(int i = 0;i<n;++i){
	for(int j = i;j<n;++j){
		int sum = 0;
		sum += array[b];
		best = max(best, sum);
	}
}
cout<<best<<endl;
```
#### algorithm 3 n
```cpp
int best = 0, sum = 0;
for(int i = 0;i < n; ++i){
	sum = max(array[k], sum+array[k]);
	best = max(best, sum);
}
cout<<best<<endl;
```
